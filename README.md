PG Gesso Drywall - Site Institucional

Descrição:
Site desenvolvido para apresentar os serviços, histórico e contato da empresa PG Gesso Drywall.
Estrutura:
- index.html: Página principal do site.
- css/style.css: Estilos do site.
- js/script.js: Scripts JavaScript (rolagem suave, interações).
- img/: Imagens utilizadas no site.
- favicon.ico: Ícone do site (exibido na aba do navegador).

Como editar:
- Abra index.html em qualquer editor de código.
- As imagens podem ser substituídas na pasta /img/.
- Para ajustar estilos, edite o arquivo style.css.
- Para alterar comportamentos, edite script.js.
